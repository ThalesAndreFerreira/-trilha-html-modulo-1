Usando o Inspetor de Elementos

O Inspetor de Elementos é uma ferramenta presente nos navegadores modernos (como Chrome, Firefox e Edge) que permite visualizar e editar o código HTML e CSS de uma página em tempo real. Ele é muito útil para aprender como uma página foi construída, testar mudanças visuais rapidamente e encontrar erros no código. Para acessar, basta clicar com o botão direito em algum elemento da página e selecionar "Inspecionar" ou pressionar F12.

Estrutura Básica do HTML

A estrutura básica de um arquivo HTML define o esqueleto de uma página da web. Começa com "html" que envolve todo o conteúdo. Dentro dela, temos o "head" (onde ficam as configurações, título da aba e links para arquivos externos como CSS) e o "body" (onde está o conteúdo visível da página).

Falando sobre Tags

As tags HTML são comandos envoltos por colchetes angulares (>) que definem os elementos da página, como parágrafos, imagens, links, títulos, listas, entre outros. A maioria das tags possui uma abertura ("p") e um fechamento ("p"), e o conteúdo vai entre elas. Algumas, como "img", são chamadas de "auto-fechadas", pois não possuem conteúdo interno.

Atributos Básicos

Atributos são usados dentro das tags HTML para fornecer informações adicionais sobre um elemento. Eles funcionam como configurações. Por exemplo, na tag "a", o atributo href indica o destino do link: a href="https://example.com">Clique aqui. Outros atributos comuns incluem src (em imagens), alt, id, class, title, entre outros.

Executando uma página HTML

Para executar uma página HTML, basta salvar o código com a extensão .html (por exemplo, index.html) e abrir esse arquivo com um navegador de internet, como o Chrome. Não é necessário nenhum programa especial. Usando Live Server o navegador interpreta o código HTML e exibe o conteúdo como uma página web.

Textos

O HTML possui várias tags para organizar e exibir textos, como "p" para parágrafos, "h1" até "h6" para títulos (do maior ao menor), "strong" para destaque em negrito, "em" para ênfase em itálico, entre outras. Essas tags ajudam a dar estrutura e significado ao conteúdo textual da página.

Listas ordenadas e não ordenadas

Listas ordenadas são aquelas que usam números para indicar ordem (com a tag "ol"), enquanto as listas não ordenadas usam marcadores (com a tag "ul"). Cada item da lista é colocado dentro da tag "li".

Voltar

Links

Os links em HTML são criados com a tag "a", usando o atributo href para definir o destino. Eles permitem navegar entre páginas e sites. Por exemplo: a href="https://google.com">Ir para o Google. É possível também usar target="_blank" para abrir o link em uma nova aba.
